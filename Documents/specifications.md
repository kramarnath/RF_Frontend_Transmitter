# Transmitter Specifications

## Standard: Bluetooth Low Energy (BLE 4.0)

| Parameter | Specification |
|---|---|
| Frequency Range | 2400 – 2483.5 MHz |
| Channels | 40 channels × 2 MHz spacing |
| Modulation | GFSK (index ≈ 0.5, ±250 kHz deviation) |
| Data Rate | 1 Mbps (mandatory), 2 Mbps (BT 5.x) |
| Output Power | 0 to +10 dBm (max +20 dBm per ETSI/FCC) |
| Transmit Current | 5 – 20 mA |
| Output Impedance | 50 Ω |
| Supply Voltage | 0.9 V |
